
# PLAN.md — PGH.ST Trash Pickup → Google Calendar Reminders

## Project Overview

A self-hosted, zero-cost system that queries the [PGH.ST](https://pgh.st) API for Pittsburgh trash/recycling/yard waste collection schedules and automatically creates Google Calendar events with reminders. Runs on GitHub Actions cron with Google Service Account auth. All secrets stored in GitHub Secrets.

---

## Architecture

```
┌─────────────┐     ┌──────────────────┐     ┌──────────────────┐
│  GitHub      │────▶│  main.py         │────▶│  Google Calendar │
│  Actions     │     │                  │     │  API (v3)        │
│  (cron: Sun) │     │  1. Query PGH.ST │     │                  │
│              │     │  2. Compute dates │     │  Events with     │
│              │     │  3. Create events │     │  2 reminders:    │
└─────────────┘     └──────────────────┘     │  - 8 PM night    │
                           │                  │    before         │
                                                      ▼                  │  - 5 AM morning  │
                                                                          ┌──────────────────┐     │    of pickup      │
                                                                                              │  PGH.ST API      │     └──────────────────┘
                                                                                                                  │  /locate/{house} │
                                                                                                                                      │  /{street}/      │
                                                                                                                                                          └──────────────────┘
                                                                                                                                                          ```

                                                                                                                                                          ## Tech Stack

                                                                                                                                                          - **Language:** Python 3.12
                                                                                                                                                          - **CI/CD:** GitHub Actions (free tier, cron schedule)
                                                                                                                                                          - **APIs:** PGH.ST `/locate/` endpoint, Google Calendar API v3
                                                                                                                                                          - **Auth:** Google Cloud Service Account (JSON key in GH Secrets)
                                                                                                                                                          - **Dependencies:** `requests`, `google-api-python-client`, `google-auth`
